### 题目描述

小明也有个怪癖，不到万不得已，从不刷卡，直接现金搞定。

#include<bits/stdc++.h>
using namespace std;
int main()
{
  vector <double> pri={180.90,10.25,56.14,104.65,100.30,297.15,26.75,130.62,240.28,270.62,115.87,247.34,73.21,101.00,79.54,278.44
