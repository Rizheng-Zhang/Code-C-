### 题目描述

如下图所示，小明用从 1 开始的正整数“蛇形”填充无限大的矩阵。<br>
<br>1 2 6 7 15 ...<br>
3 5 8 14 ...<br>
4 9 13 ...<br>
10 12 ...<br>
11 ...<br>
...<br>
<br>容易看出矩阵第二行第二列中的数是 5。请你计算矩阵中第 20 行第 20 列的数是多少？

### code
```c++
#include <iostream>
using namespace std;
int main()
{
  int ans=1;//ans[1]
  for(int i=1;i<20;i++){
    /*找数字规律，对角线（1,1）(2,2)......依次为1,5,13,25...
    每次多差一个四（4,8,12...）所以，对角线第1个数ans[1]=1,第i个数ans[i]=ans[i-1]+4*(i-1)
    第20个数ans[20]=ans[19]+4*(19)  */
    ans+=i*4;//循环迭代
  }
  cout<<ans;
  return 0;
}
```
